💡 What: Added keyboard shortcut hints (Left), (Right), (Esc) to the navigation and exit buttons in the Sharpness Tool.
🎯 Why: Users often miss that actions like Next, Previous, and Exit Focus Mode have global keyboard shortcuts unless those shortcuts are explicitly hinted in the corresponding UI button text.
📸 Before/After: Visual regression baselines have been updated to reflect the button label changes.
♿ Accessibility: Improves discoverability of keyboard navigation, helping users who rely on keyboard shortcuts.
