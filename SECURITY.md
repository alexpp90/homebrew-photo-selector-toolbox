# Security Policy

## Supported Versions

Only the latest stable release of the Photo Selector Toolbox is supported for security updates.

| Version | Supported |
| ------- | --------- |
| >= 0.2.0 | :white_check_mark: Yes |
| < 0.2.0  | :x: No |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it responsibly by contacting the project maintainers directly via email. **Do not create a public GitHub issue for security issues.**

Please email your report to the contact address associated with the project's GitHub organization or repository owners.

In your report, please include:
- A description of the vulnerability and its potential impact.
- Detailed steps to reproduce the issue (including any proof-of-concept scripts or commands).
- The operating system and version of the Photo Selector Toolbox you are using.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible. Once resolved, we will publish a new release containing the fix and credit you for the discovery (unless you prefer to remain anonymous).
