## 2024-05-18 - `os.walk` Mocking in Tests
**Learning:** When replacing `Path.rglob` with `os.walk` for highly optimized directory traversal, old test mocks designed for `rglob` will cause `AssertionError`s. Specifically, test suites usually mock `Path.rglob` to return a list of `Path` objects. However, `os.walk` expects a generator yielding tuples of `(dirpath, dirnames, filenames)`.
**Action:** When refactoring traversal methods, actively search the test suite (`grep -rn "rglob" tests/`) and rewrite the mocks to properly simulate `os.walk` using a helper function that yields the necessary tuples.
