## 2024-05-19 - Added Delete Keyboard Shortcut to Duplicate Finder
**Learning:** Keyboard shortcuts vastly improve the usability of repetitive actions like deleting duplicate files, but their bindings must be mapped carefully in a multi-tab application (e.g., ensuring `winfo_ismapped()` is checked) to prevent actions from triggering unintentionally while focused on other tabs.
**Action:** When adding global shortcuts to child components in a Tkinter notebook, wrap the binding logic with visibility checks, guard against text input interference, and use `try/except` to protect test suites from mocked root windows. Ensure the shortcut is also documented directly on the UI button.
